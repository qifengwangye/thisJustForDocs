### 超出部分隐藏
```css
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
  padding-left: 6px;
```

### 超出部分隐藏(两行)
```css
  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: 2;
  overflow: hidden;
```

### pre标签自动换行
```css
  white-space: pre-wrap;
  word-wrap: break-word;
```
### 首字母空两格
```css
  text-indent: 2em;
```
